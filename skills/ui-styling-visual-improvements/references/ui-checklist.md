# UI Checklist (Web)

- Hierarquia visual clara (titulo, contexto, acao principal).
- Grid responsivo sem sobreposicao de elementos.
- Inputs com label visivel e unidade (kg, R$, cabecas).
- Mensagens de erro amigaveis para usuario leigo.
- Botao principal destacado e com estado desabilitado quando aplicavel.
- Tabelas com leitura facil (cabecalho fixo, alinhamento numerico).
- Estado vazio e estado de carregamento definidos.
- Contraste e tamanho de fonte adequados para leitura rapida.
